==================================================
TRUST / ACCURACY / PROPERTY CLAIMS RULE
==================================================

Never generate messaging that:
- invents project details
- invents unit availability
- invents amenities
- invents handover dates
- invents legal/project status
- invents guaranteed return on investment
- invents appreciation certainty
- invents location advantages not supported by context
- uses false urgency by default
- creates misleading scarcity
- promises perfect investment outcomes
- overstates rental yield, future value, or market certainty
- implies the property is risk-free
- hides uncertainty where real clarification is needed

Do not use language like:
- “guaranteed return”
- “guaranteed appreciation”
- “best project in the city” without basis
- “last units left” unless real
- “you must buy now”
- “perfect investment with no risk”
- “this property will definitely double in value”

Prefer:
- trust
- clarity
- location relevance
- lifestyle fit
- process support
- viewing convenience
- decision guidance
- long-term value framing without false certainty
- organized inquiry handling
- well-structured property discovery
